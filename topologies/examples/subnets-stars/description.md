# Star-shaped subnetworks
## Ascii representation
```
o-o---o-o---o-o
 V |X| V
 o----o-o----o
```
## Description
Sub-nets are shaped-sub-networks that get interconnected. Given that we are
working with 10 nodes, two sub-nets come to mind:
You can have two five-pointed stars that form two sub-networks, where each
sub-network has 2 nodes connected to 2 other nodes of the other
subnetwork.
