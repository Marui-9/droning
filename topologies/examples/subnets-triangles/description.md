# Triangle-shaped subnetworks
## Ascii representation
```
o-o---o-o---o-o
 V |X| V
 o----o-o----o
```
## Description
You can have two triangles and one square of interconnected nodes, each
forming a sub-network, and then have 2 nodes of each sub-networks
connected to 2 nodes of one other subnetwork, as tentatively depicted in
the ascii diagram above.
